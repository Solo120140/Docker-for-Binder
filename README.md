# webchain-miner-dockerized
webchain-miner container for WebChain Network


## To build:
```
docker build -f Dockerfile -t webchainminer .
```

## To run the miner:
```
docker run webchainminer -o webchain.wattpool.net:3333 -u  WALLET -p x
```
